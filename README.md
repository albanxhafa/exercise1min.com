# exercise1min.com

Landing page for **Exercise 1 Min** — a fully-offline mobile app for one-minute movement
breaks (English / global). Companion Albanian site: [ushtrohu.al](https://ushtrohu.al).

Static site — plain HTML, CSS, and JS with no build step, frameworks, or external assets.
Deployed to **GitHub Pages** on every push to `main` via `.github/workflows/deploy.yml`.

## Local preview

```bash
python3 -m http.server
# then open http://localhost:8000
```
